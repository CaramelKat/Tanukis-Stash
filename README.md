The worlds first e621 client for iOS, iPadOS, and macOS built with SwiftUI

[Get the Latest Release Here](https://github.com/caramelkat/tanukis-stash/releases/latest)

### Features
- Infinite scrolling
- Post details page
- Search for tags
- Supports most media formats
